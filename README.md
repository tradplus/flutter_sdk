# TradPlus flutter sdk

Tradplus广告聚合平台 flutter插件 

支持平台：iOS , Android

支持类型包括： 激励视频 , 插屏广告, 原生广告, 横幅广告, 开屏广告

接入指南 参考[官方文档](https://docs.tradplusad.com/docs/integration_flutter/flutter_integration_base/sdk_config)
